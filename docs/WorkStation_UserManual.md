## 1. หน้าเลือก Station

เมื่อเข้าสู่เมนู **SETUP / REFURBISH** จะพบกับหน้าจอเลือก Station ซึ่งแสดงเป็นรูปหกเหลี่ยม (Hex)

### Station ทั้งหมด

| ลำดับ | Station | รหัส |
|:-----:|---------|:----:|
| - | Work Order List | - |
| - | Stock Report | - |
| 1 | Inspector | 0010 |
| 2 | Remove Part | 0020 |
| 3 | Clean | 0030 |
| 4 | Color | 0040 |
| 5 | Fix Cooling | 0050 |
| 6 | Assembly Part | 0060 |
| 7 | Test | 0070 |
| 8 | QC Packing | 0080 |

### วิธีใช้งาน
- **กดที่ Station** ที่ต้องการ → ระบบจะพาไปยังหน้ารายการ Work Order ที่อยู่ใน Station นั้น
- **กดที่ Work Order List** → ดูรายการ Work Order ทั้งหมดและจ่ายงาน
- **กดที่ Stock Report** → ดูรายงานอะไหล่คงเหลือ

## 2. หน้ารายการ Work Order

หลังจากเลือก Station จะเข้าสู่หน้าตารางรายการ Work Order

### ข้อมูลที่แสดง
| คอลัมน์ | รายละเอียด |
|---------|-----------|
| SLA Timer | แสดงเวลา SLA ที่เหลือ |
| Work Order | หมายเลข Work Order |
| Order Type | ประเภทงาน |
| Production Start Date | วันที่เริ่มผลิต |
| Equipment | รหัสเครื่อง |
| Start Date / Finish Date | วันเริ่มต้น / วันสิ้นสุดงาน |
| Description | รายละเอียดงาน |

### สถานะ SLA
สามารถกรองสถานะ SLA ได้จาก dropdown:
- 🟢 **เขียว** — เวลาเหลือเยอะ
- 🟡 **เหลือง** — ใกล้ถึงเวลา
- 🔴 **แดง** — เลยเวลา SLA แล้ว

### วิธีใช้งาน
1. **กรอง SLA Status** — เลือกจาก dropdown เพื่อกรองสถานะ
2. **ค้นหา** — พิมพ์ในช่องค้นหาเพื่อหา Work Order
3. **กดที่แถว** → เข้าสู่หน้าปฏิบัติงาน (Work Station)
4. **จ่ายงาน** (เฉพาะหน้า Work Order List) — เลือก checkbox หลายรายการ แล้วกดปุ่ม "จ่ายงาน"

## 3. หน้าปฏิบัติงาน (Work Station)

หน้าหลักสำหรับปฏิบัติงาน แบ่งเป็น **4 แท็บ**:

### 3.1 แท็บ Work Order (แท็บแรก)

แสดงข้อมูลหลักของงานและปุ่มการทำงานทั้งหมด

#### ข้อมูลที่แสดง
- **Checklist Station** — checkbox สำหรับติ๊กเลือก station ที่ต้องทำงาน
- **นาฬิกาจับเวลา (Timer)** — แสดงเวลาที่ใช้ทำงาน
- **SLA Time** — เวลา SLA ที่กำหนด
- **Start Date / Finish Date** — วันเริ่มต้นและสิ้นสุด
- **Start Time / Finish Time** — เวลาเริ่มต้นและสิ้นสุด

#### ปุ่มการทำงานหลัก

| ปุ่ม | สี | การทำงาน |
|------|:--:|---------|
| **Start** | 🟢 เขียว | เริ่มทำงาน — ระบบจะถามให้เลือกพนักงานก่อน แล้วจับเวลาเริ่ม |
| **Finish** | 🔵 น้ำเงิน | เสร็จสิ้นงาน — บันทึกเวลาสิ้นสุดและ check station ที่เลือก |

> **หมายเหตุ:** ปุ่ม Start จะแสดงเมื่อยังไม่ได้เริ่มงาน, ปุ่ม Finish จะแสดงเมื่อกำลังทำงานอยู่

#### ปุ่มการทำงานรอง

| ปุ่ม | สี | การทำงาน |
|------|:--:|---------|
| **Check List** | 🔵 น้ำเงิน | เปิด Checklist สำหรับกรอกรายละเอียดงาน (เฉพาะ Station Inspector, Clean, Test) |
| **Completed** | 🟣 ม่วง | ปิดงาน (มีให้เลือกปกติหรือไม่ปกติ) |
| **Remark** | ⬛ เทา | เพิ่มหมายเหตุ เช่น Delay, Waiting Part, Rework |
| **Return** | 🔴 แดง | ส่งงานคืนไปยัง Station ก่อนหน้า |
| **QC Return** | 🔴 แดง | QC ส่งงานคืนไป Station ที่ต้องแก้ไข |

---

#### 🟣 รายละเอียดปุ่ม Completed

เมื่อกดปุ่ม **Completed** ระบบจะให้เลือก:

**เส้นทาง "ปกติ"**
1. ระบบดึงข้อมูล Close Type จาก Server
2. แสดง Popup ยืนยันข้อมูล (Work Order, Close Type, เวลาเริ่ม, เวลาสิ้นสุด)
3. กด **ยืนยัน** → ระบบบันทึกปิดงานและส่งข้อมูลไปยัง Server

**เส้นทาง "ไม่ปกติ"**
1. ระบบแสดง Dropdown ให้เลือกประเภทการปิดงาน (ขึ้นอยู่กับ Order Type)
2. แสดง Popup ยืนยันข้อมูล
3. กด **ยืนยัน** → ระบบบันทึกปิดงานและส่งข้อมูลไปยัง Server

---

#### 🔴 รายละเอียดปุ่ม Return / QC Return

1. ระบบแสดง Dropdown ให้เลือก Station ที่ต้องการส่งคืน (แสดงเฉพาะ Station ที่ผ่านมาแล้ว)
2. เลือกเหตุผลในการส่งคืน: **Delay** / **Waiting Part** / **Rework**
3. กด **Confirm** → ระบบส่งงานคืนไปยัง Station ที่เลือก

### 3.3 แท็บ Sparepart

แสดงรายการอะไหล่ที่ใช้ใน Work Order

| การทำงาน | วิธีใช้ |
|---------|--------|
| ดูรายการอะไหล่ | ดูจากตาราง |
| แก้ไขจำนวน | กดที่ปุ่มแก้ไข → ใส่จำนวนใหม่ → กด Save |
| แก้ไขรายการอะไหล่ | กดปุ่ม **"แก้ไขรายการอะไหล่"** → ไปหน้าจัดการอะไหล่ |

---

### 3.4 แท็บ Upload Image

อัปโหลดรูปภาพประกอบ Work Order

| ขั้นตอน | รายละเอียด |
|---------|-----------|
| 1 | ระบบแสดงรายการรูปภาพที่ต้องอัปโหลด (ตาม Master Template) |
| 2 | กดเลือกรูปภาพสำหรับแต่ละรายการ |
| 3 | กดปุ่ม **"บันทึกรูปภาพทั้งหมด"** เพื่ออัปโหลดทุกรูปพร้อมกัน |

> รูปภาพที่เคยอัปโหลดแล้วจะแสดงรูปเดิม สามารถเลือกรูปใหม่เพื่อแทนที่ได้

## 4. หน้าจัดการอะไหล่

เข้าถึงได้จากแท็บ Sparepart → ปุ่ม "แก้ไขรายการอะไหล่"

- เพิ่ม/ลบ อะไหล่
- แก้ไขจำนวนอะไหล่

## 5. หน้ารายงาน Stock

เข้าถึงได้จากหน้า Setup / Refurbish → กดที่ **Stock Report**

- ดูรายงานอะไหล่คงเหลือ
- กดเข้าดูรายละเอียดแต่ละรายการ

---

## ⚠️ สิ่งที่ควรระวัง

1. **ต้อง Start ก่อน Finish** — ไม่สามารถกด Finish ได้หากยังไม่ได้กด Start
2. **เลือกพนักงานก่อนเริ่มงาน** — เมื่อกด Start ระบบจะให้เลือกพนักงานที่ทำงาน
3. **Checklist ติ๊กก่อน Finish** — ควรติ๊ก Checklist Station ให้ครบก่อนกด Finish
4. **Return ได้เฉพาะ Station ก่อนหน้า** — ไม่สามารถ Return ไปยัง Station ที่อยู่ข้างหน้าได้
5. **อัปโหลดรูปภาพ** — ตรวจสอบให้ครบทุกรูปก่อนกดบันทึก

---

## 🔄 ลำดับขั้นตอนการทำงานแนะนำ

```
เลือก Station → เลือก Work Order → กด Start (เลือกพนักงาน)
    → ทำงาน → ติ๊ก Checklist / กรอก Check List
    → อัปโหลดรูปภาพ
    → กด Finish
    → กด Completed (ปิดงาน)
```
