WorkStation & Refurbish

คู่มือการใช้งานและรายละเอียดฟังก์ชันสำหรับเจ้าหน้าที่ปฏิบัติงาน ครอบคลุมตั้งแต่นำเข้างาน การเบิกอะไหล่ ไปจนถึงการปิดงานและตรวจสอบคุณภาพ

PHASE 1: ACCESS POINT

Setup & Refurbish Hub

จุดเริ่มต้นของการปฏิบัติงาน (Entry Point) ออกแบบมาเพื่อให้เจ้าหน้าที่สามารถเลือกสถานีงาน (Work Station) ได้อย่างรวดเร็วและแม่นยำ ผ่านอินเทอร์เฟซแบบกราฟิกที่เข้าใจง่าย

Functional Overview

หน้าจอนี้ทำหน้าที่เป็น Central Hub สำหรับการ Routing ผู้ใช้งานไปยังกระบวนการที่ถูกต้อง โดยมีการจัดวาง Layout แบบ Hexagonal Grid เพื่อรองรับจำนวน Station ที่อาจเพิ่มขึ้นในอนาคต และยังมีระบบ Visual Navigation โดยใช้ไอคอนสื่อความหมายชัดเจน ทำให้พนักงานสามารถเข้าถึงเมนูที่ต้องการได้อย่างรวดเร็ว

ข้อดีหลักของ Hexagonal Layout: ช่วยให้ปุ่มแต่ละปุ่มมีพื้นที่กว้างขึ้น เหมาะสำหรับการทำงานบน Tablet หรือ Touch Screen ในสภาพแวดล้อมโรงงาน โดยสามารถเพิ่ม Station ใหม่ได้โดยไม่กระทบต่อ Layout เดิม และยังช่วยลดความผิดพลาดจากการกดผิดปุ่ม

  • Authentication & Authorization: ระบบตรวจสอบสิทธิ์พนักงานก่อนเข้าถึงเมนูต่างๆ เชื่อมต่อกับ Active Directory และบันทึก Audit Log เพื่อความปลอดภัย
  • Responsive Layout: รองรับการใช้งานผ่าน Tablet (10-12 นิ้ว), Desktop และมาตรฐานหน้าจออุตสาหกรรม ปรับขนาดอัตโนมัติตาม Resolution
  • Easy Navigation: ลดความซับซ้อนในการเข้าถึงเมนูด้วย Graphic-based UI ใช้ไอคอนและสีแยกประเภทงาน ช่วยให้พนักงานใหม่เรียนรู้ได้เร็วขึ้น

Available Work Stations

01
Inspector & Incoming Check
ตรวจสอบสภาพสินค้าขาเข้า บันทึกความเสียหายเบื้องต้น
02
Disassembly & Remove Parts
ขั้นตอนการถอดชิ้นส่วนและแยกประเภทอะไหล่
03
Cleaning & Preparation
ทำความสะอาดชิ้นงาน เตรียมพื้นผิว
04
Assembly, QC & Finishing
ประกอบงาน ตรวจสอบคุณภาพ และส่งมอบ

Presentation Tip

ควรอธิบายเพิ่มเติมว่า "Hex Grid" ช่วยให้กดปุ่มได้ง่ายบนหน้าจอสัมผัส (Touch Screen Friendly) และรองรับการขยาย Station ใหม่ๆ ได้โดยไม่เสีย Layout

PHASE 2: MANAGEMENT VIEW

Dashboard Refurbish

ศูนย์รวมรายการใบงาน (Work Orders) ที่รอการดำเนินการ พร้อมเครื่องช่วยจัดลำดับความสำคัญอัจฉริยะ (SLA Priority)

Intelligent Work List

ระบบแสดงรายการใบงานแบบ Real-time โดยดึงข้อมูลจาก SAP/ERP เชื่อมต่อผ่าน API Gateway ช่วยให้หัวหน้างานและช่างเทคนิคเห็นภาพรวมของงานที่ค้างอยู่ (Backlog) ทั้งหมด และสามารถตัดสินใจเลือกทำงานได้ทันที โดยคำนึงถึง Priority และ SLA ของแต่ละงาน

จุดเด่น: Dashboard นี้ช่วยลดปัญหาการหาใบงาน (Work Order) ที่เคยต้องใช้กระดาษหรือ Excel โดยมีการ Sync ข้อมูลทุก 30 วินาที และแสดงเฉพาะงานที่ยังไม่เสร็จสมบูรณ์ พร้อมไฮไลต์งานที่ใกล้เกิน SLA

Advanced Filtering

กรองงานตามสถานะ SLA (Normal, Warning, Overdue) หรือค้นหาด้วยเลข Work Order

SLA Countdown

แสดงเวลาที่เหลือก่อนกำหนดส่งมอบงาน (Countdown Timer) ช่วยกระตุ้นให้ทำงานเสร็จตามเวลา

Quick Information

แสดงรายละเอียดสำคัญทันที เช่น Equipment, Order Type และ Description โดยไม่ต้องกดเข้าไปดูข้างใน

User Interaction

ผู้ใช้งานสามารถ คลิกที่แถว (Row Click) ของรายการใบงาน เพื่อเข้าสู่หน้าจอลงรายละเอียดการปฏิบัติงาน (Execution Screen) ในขั้นตอนถัดไป

PHASE 3: ACTION & EXECUTION

WorkStation Execution

หน้าจอการทำงานหลักที่รวมทุกฟังก์ชันไว้ในจุดเดียว เพื่อให้ช่างเทคนิคปฏิบัติงานได้ต่อเนื่องโดยไม่ต้องเปลี่ยนหน้าจอ

Interface Showcase

แสดงตัวอย่างหน้าจอจริงจากระบบที่พัฒนาขึ้น ครอบคลุมทุกฟีเจอร์หลัก

1

Job Details & SLA

หน้าจอแสดงข้อมูลงาน พร้อม Timer นับถอยหลังเพื่อควบคุม SLA แบบ Real-time

2

Employee Team

จัดการทีมงาน เพิ่ม-ลบพนักงาน พร้อมระบุบทบาท (Leader/Technician)

3

Evidence Upload

อัปโหลดภาพหลักฐาน Before/After พร้อมกำหนด Master Images ที่บังคับถ่าย

4

Spare Parts Inventory

แสดงรายการอะไหล่คงคลังที่สามารถเบิกใช้งานได้ทันที พร้อมระบุตำแหน่งจัดเก็บ (Bin Location) และจำนวนคงเหลือแบบ Real-time เชื่อมต่อกับระบบ Master Data

5

Parts Request Cart

ระบบตะกร้าเบิกอะไหล่ที่ช่วยรวบรวมรายการที่ต้องการเบิกในครั้งเดียว ลดขั้นตอนเอกสารซ้ำซ้อน และเชื่อมต่อกับระบบตัดสต็อกอัตโนมัติเมื่อได้รับการอนุมัติ

1

Job Details & SLA

แสดงข้อมูล Work Order, Equipment และเวลานับถอยหลัง (Timer) เพื่อควบคุม SLA อย่างเคร่งครัด

2

Employee Management

จัดการรายชื่อพนักงานที่ปฏิบัติงานใน Job นั้นๆ รองรับการทำงานแบบทีม (Leader/Technician)

3

Spare Parts Inventory

แท็บสำหรับเบิก-จ่ายอะไหล่ ตัดสต็อกจริง (Stock Deduction) และบันทึกยอดการใช้งานตามจริง

4

Evidence Upload

รองรับการอัปโหลดรูปภาพหลักฐาน (Before/After) โดยมีการกำหนด Master Images ที่จำเป็นต้องถ่าย

PHASE 4: INVENTORY MANAGEMENT

Stock Report System

ระบบจัดการคลังอะไหล่และการขอเบิกวัสดุ เพื่อควบคุมสต็อกและติดตามการใช้งาน Real-time

Stock Management Interface

ระบบจัดการคลังอะไหล่ครบวงจร ตั้งแต่การเบิก ตรวจสอบ จนถึงการอนุมัติ

1

Stock Report List

หน้าจอแสดงรายการอะไหล่ทั้งหมดในคลัง พร้อมข้อมูล Stock Level, Location และสถานะพร้อมใช้งาน

2

Cart & Request System

ระบบตะกร้าสำหรับรวบรวมอะไหล่ที่ต้องการเบิก พร้อมระบุจำนวน หน่วย และ Work Order ที่เกี่ยวข้อง

3

Item Details & History

แสดงรายละเอียดครบถ้วนของอะไหล่ ประวัติการเบิก-จ่าย, ตำแหน่งจัดเก็บ และข้อมูล Specification

Real-time Inventory

เชื่อมต่อกับระบบ SAP เพื่อดึงข้อมูล Stock แบบ Real-time รวมถึงสถานะ Reservation

Approval Workflow

ระบบอนุมัติการเบิกหลายระดับ ตามจำนวนและมูลค่า พร้อมแจ้งเตือนผู้เกี่ยวข้อง

Usage Analytics

วิเคราะห์การใช้งานอะไหล่ เพื่อคาดการณ์ความต้องการและวางแผนจัดซื้อ

PHASE 5: QUALITY ASSURANCE

QC Dashboard

ระบบตรวจสอบคุณภาพและจัดการงาน QC แบบ Real-time พร้อมติดตามสถานะงานในแต่ละสถานี

QC Management Interface

แดชบอร์ดสำหรับติดตาม Order ที่อยู่ในกระบวนการ QC และจัดการงานตรวจสอบคุณภาพ

QC Order Tracking

หน้าจอแสดงรายการ Work Order ที่อยู่ในกระบวนการ QC พร้อมสถานะการตรวจสอบและผู้รับผิดชอบ

Quality Inspection

ตรวจสอบคุณภาพตามมาตรฐาน พร้อมบันทึกผลการตรวจสอบและรูปภาพหลักฐาน

Order Status Tracking

ติดตามสถานะ Order ใน QC แบบ Real-time รู้ว่างานไหนผ่าน ไม่ผ่าน หรือรอตรวจ

Defect Management

บันทึกและจัดการข้อบกพร่องที่พบ พร้อมระบบ Reject & Rework
