คู่มือการใช้งานระบบพิมพ์ QR Code

ระบบสำหรับค้นหาและเลือกรายการ Work Order เพื่อนำมาพิมพ์เป็นสติ๊กเกอร์ QR Code สำหรับติดชิ้นงาน

1. ส่วนประกอบของหน้าจอ (Interface Overview)

หน้าจอแบ่งออกเป็น 2 ส่วนหลัก:

  • แผงควบคุม (Control Panel): อยู่ด้านซ้าย (หรือด้านบนในมือถือ) สำหรับค้นหาและเลือกรายการ
  • พื้นที่แสดงผล (Preview Area): อยู่ด้านขวา (หรือด้านล่าง) สำหรับแสดงรายการสติ๊กเกอร์ที่จะพิมพ์

คำอธิบายเครื่องมือ

  • ช่องค้นหา (Search Box): ใช้กรองรายการ โดยพิมพ์ Work Order ID, ชื่อ Station, หรือรหัส Operation ระบบจะแสดงรายการที่เกี่ยวข้องทันที
  • ปุ่มล้างคำค้นหา (Clear Button): ไอคอนกากบาท (X) ในช่องค้นหา เพื่อลบข้อความทั้งหมดในคลิกเดียว
  • Checkbox เลือกทั้งหมด: ใช้เลือกหรือยกเลิกการเลือกรายการทั้งหมดที่แสดงอยู่ (ตามผลการค้นหา)
  • ปุ่มสั่งพิมพ์ (Print Button): ปุ่มสีน้ำเงิน "สั่งพิมพ์ (Browser Print)" สำหรับเปิดหน้าต่างพิมพ์ของเบราว์เซอร์

2. วิธีการใช้งาน (How to Use)

1 ค้นหารายการที่ต้องการ

พิมพ์รหัสใบงาน (Work Order) หรือชื่อสถานีงานลงใน ช่องค้นหา รายการที่ไม่เกี่ยวข้องจะถูกซ่อนไป

2 เลือกรายการพิมพ์

คลิก Checkbox หน้าชื่อรายการที่ต้องการพิมพ์ หรือคลิก "เลือกทั้งหมด" ด้านบนเพื่อเลือกทุกรายการที่ค้นเจอ

Tip: รายการที่เลือกจะปรากฏขึ้นมาเป็นตัวอย่างสติ๊กเกอร์ด้านล่าง/ด้านขวา
3 ตรวจสอบความถูกต้อง

ดูที่หน้าจอ Preview ว่าจำนวนสติ๊กเกอร์และข้อมูลบนสติ๊กเกอร์ถูกต้องหรือไม่ (เช่น QR Code, รหัส Order, ชื่อ Station)

4 สั่งพิมพ์

กดปุ่ม สั่งพิมพ์ (Browser Print) ระบบจะเปิดหน้าต่าง Print ของ Browser ขึ้นมา
